## My process

I inspect first the design on Figma Dev mode. 
I write html and copy css properties from figma.
I ask Github copilot on vscode to review my code and suggest improvement.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- Mobile-first workflow

### Continued development

I would like to improve on the best practice of writing modern css, using variables, css grid, flex, nesting etc.


### AI Collaboration

I usually use the autocomplete functionality in VSCode via Github Copilot. I accept if it makes sense but to challenge myself to learn I will write my own code let copilot review and suggest improvement. If I am stuck I will ask questions on how to do certain stuff and what is the best practice.

It easier now to apply the exact design properties on the final design because of Figma Dev mode, you can just copy and paste. But sometimes there's wrong property (e.g. instead of using var for font-text-preset it uses the var for --spacing-100) that's why you should still ask AI to review the code first before pushing to github.

- Tools that I used (ChatGPT, GitHub Copilot)
